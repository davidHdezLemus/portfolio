Portfolio

Task 1

  • Aplicación
  • Código
  • Descripción:

    Cuando entres en la URL, verás un contador de tiempo centrado en la página que muestra el tiempo restante hasta una fecha específica. El color del contador cambia según el tiempo. Puedes seleccionar una nueva fecha, y cuando llegue a cero, aparecerá un mensaje de feliz cumpleaños.

Task 2

  • Aplicación
  • Código
  • Descripción:

    Cuando entres en la URL, verás un juego de cartas "Exploding Cards". Tendrá un título, un área para dibujar cartas, botones para jugar y reiniciar, y sonidos de fondo y efectos. El juego consiste en dibujar cartas de un mazo hasta que se acaben. Es un juego interactivo y dinámico que ofrece una experiencia de juego completa.

Task 3

  • Aplicación
  • Código
  • Descripción:

    Esta aplicación es una herramienta para gestionar gastos compartidos entre varios usuarios. Permite añadir usuarios con sus respectivos géneros e iconos, registrar gastos, calcular los balances individuales, y visualizar la lista de usuarios, los gastos registrados y los balances.


    Además, ofrece la opción de saldar cuentas automáticamente.
